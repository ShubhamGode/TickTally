
# TickTally

**Track time. Count tasks. Stay productive.**
TickTally is a simple and modern web application that helps users track both **time** and **quantity** of their daily activities through an intuitive interface.

---
## 🚀 Features

- ⏱️ **Time Tracker** – Monitor how much time you spend on tasks  
- 🔢 **Quantity Tracker** – Track counts, repetitions, or quantities  
- 🎨 **Modern UI** – Clean dark theme with smooth user experience  
- ⚡ **Fast & Lightweight** – Built using pure HTML, CSS, and JavaScript  
- 🧭 **Easy Navigation** – Simple selection between tracking modes  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript
